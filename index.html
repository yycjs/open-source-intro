jQuery

  • Simplifies DOM manipulation, traversal and event handling
  • AJAX support
  • Effects
  • UI elements with jQuery UI (~200Kb)

  • Plugin mechanism

  • ~ 32Kb in size

jQuery - Example

jQuery

    $('button').click(function() {
        var name = $('[name="your-name"]').val();
